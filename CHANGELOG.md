# Change Log

## [1.1.0] - 2026-03-15

### Added
- **Narvo Dusk** — Fourth theme variant: near-black warm background (#14100C), amber accents (#FFBA4A), amber borders (#785834), warm gold text tiers. Derived from actual `[data-theme='dusk']` CSS in the Narvo News codebase.

## [1.0.0] - 2026-03-15

### Added
- **Narvo Nature** — Default dark theme: deep forest background, sand/beige accents, sage green syntax
- **Narvo Sun** — Light theme: warm parchment background, deep green accents, editorial clarity
- **Narvo Moon** — Dark theme: deep navy background, soft purple accents, Vite-inspired palette
- Semantic category colors from Narvo's news taxonomy (finance, tech, urgent, health, etc.)
- Swiss-brutalist sharp borders throughout workbench
- Comprehensive syntax highlighting for JS/TS, Python, Rust, HTML/CSS, JSON, Markdown
- Semantic token colorization support
- Bracket pair colorization with category-mapped palette
